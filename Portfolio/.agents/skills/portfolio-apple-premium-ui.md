# portfolio-apple-premium-ui

## Purpose
Raise interface quality to Apple-inspired premium level: calm, confident, expensive-looking, and precise.

## When to use
- Hero redesigns and first-screen composition work.
- Global visual polish and premium quality passes.
- Creating new premium sections (showcase, process, featured projects, case blocks).
- Fixing UI that feels generic, heavy, or “template-like”.

## Core rules
- Build composition first, decoration second.
- Keep layouts clean and balanced:
  - Strong focal point.
  - Controlled negative space.
  - Clear visual entry path.
- Use layered surfaces and restrained highlights instead of loud gradients.
- Keep typography premium:
  - Avoid noisy font mixing.
  - Use weight/size/spacing to communicate confidence.
- Make controls tactile:
  - Slight motion on hover.
  - Clear focus ring.
  - No abrupt state changes.
- Keep copy and UI aligned: premium visuals require premium wording.
- Make hero sections feel intentional:
  - Brand cue.
  - Strong value signal.
  - Immediate action path.
- Maintain calm color confidence:
  - Accent color should guide attention, not dominate.
  - No neon chaos or washed-out gray monotony.

## Checklist
- Verify hero has a clear focal point and meaningful CTA hierarchy.
- Verify section transitions feel cohesive, not stitched together.
- Verify UI density is balanced (not cramped, not empty).
- Verify cards/components look product-grade, not placeholder-grade.
- Verify first impression in both themes feels premium within 3 seconds.

## Anti-patterns
- “Fancy” effects with weak information hierarchy.
- Overblown blur/glow that hurts readability.
- Inconsistent corners/shadows across components.
- Repetitive card grids with no depth or narrative.
- Premium claim with cheap copy and empty sections.

## Expected output
- A visually confident, high-fidelity UI pass with coherent premium direction.
- Improved hero and section composition.
- UI that looks intentionally designed, not auto-generated.

## Definition of done
- The page feels premium, calm, and complete on first view.
- Visual decisions are consistent across routes and component families.
- No section looks generic, unfinished, or stylistically off-brand.
