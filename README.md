# Portfolio

Personal portfolio website of Artem Diatlenko, built with Next.js 16, React 19, TypeScript, and Tailwind CSS 4.

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS 4

## Features

- Responsive portfolio layout (Home, About, Projects, Contact)
- Reusable UI components for header, hero, and footer
- EN/PL language switch with persisted preference
- Smooth route-change scroll behavior with reduced-motion fallback

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - run ESLint

## Notes

- Project is currently configured as static content (`next build` pre-render).
- Contact and project links point to real external profiles/repositories.
